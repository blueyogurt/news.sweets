複数のシフトも一目瞭然わかりやすい♪カレンダーアプリ『Shift Work Style』

2016.6.25 remove_red_eye2475

1日に複数の仕事をしていたり、仕事が日をまたいでしまう場合、今までのカレンダーだといまいち使いにくいと感じませんでしたか？

『Shift Work Style』は、シフト制のバイトや仕事をする方向けの、ありそうでなかったカレンダーアプリです。

まずは、項目ごとに色設定をしてシフトを登録します。

今回は「ケーキ屋」さんと「コンビニ」でアルバイトをしている学生さんを想定してみました。

『今月の目標額』も設定できます。

深夜料金や時間の設定もそれぞれ可能なので、『今月の目標額』に向けての計算をすべてお任せできます♪

画像引用元：Shift Work Style

登録済みのシフトはワンタッチで入力できる他、少し勤務時間が前後する場合は『直接入力する』をタップ。

画像引用元：Shift Work Style

項目を選んで、かんたんにシフト変更できます。

画像引用元：Shift Work Style

その日、その月の予定がひとめでわかりやすい！

土日はケーキやさんがお休みなので、メモ欄に美容院の予定をいれてみました♪

画像引用元：Shift Work Style

どの項目でどれくらい稼げているか、現在入力したシフトでは合計いくらになるのか、『給与』の項目で確認できます。

シフト制で働いている方は、年齢、仕事問わずかなり重宝しそう。
